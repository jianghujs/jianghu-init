# common

## 描述

Vue 原生变量和 uiAction 配置，包含 data、props、watch、methods、computed 等。

## 示例

```javascript
{
  "common": {
    mixins:[],
    filters: {},
    props: {},
    data: {}, // 普通变量
    dataExpression: {}, // 动态变量
    computed: {},
    watch: {},
    created: {},
    mounted: {},
    doUiAction: {},
    methods: {},
  }
}
```

## 注意事项

- 确保 common 的值符合规范
- 参考示例正确配置
