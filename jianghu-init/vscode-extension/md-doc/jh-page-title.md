## jh-page-title

页面标题，用于在用户界面中展示。

### 示例
```javascript
{ 
  tag: 'jh-page-title',               // 标签名      
  value: "我的任务",                   // 标题内容      
  attrs: { cols: 12, sm: 6, md:4 },   // 布局配置      
  helpBtn: true,                      // 帮助按钮      
  backBtn: true,                      // 返回按钮      
  slot: []                            // 插槽
}
```
